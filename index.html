列表 list
切片 lst[start : stop : step]
规则
  • 结果包含 start，不含 stop
  • 省略 start：从头；省略 stop：到尾
  • step 正：从左往右；step 负：从右往左
  • 负数下标：-1 = 最后，-2 = 倒数第二

速查
考前速查卡
有返回值 vs 无返回值

有返回值：

  • pop() → 被删除的元素
  • index(x) → 下标
  • count(x) → 次数
  • sorted(lst) → 新列表

无返回值（None）：

  • append / extend / insert
  • remove / clear
  • sort / reverse
改变 vs 不改变原列表

改变原列表：

  • sort, reverse, append
  • extend, insert, remove, pop

不改变（返回新值）：

  • sorted(lst) 返回新列表
  • lst[:] 切片返回新列表
