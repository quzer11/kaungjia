列表 list

列表切片 lst[start:stop:step]

切片规则
  • lst[start:stop] → 从 start 到 stop-1
  • lst[::step] → 按步长取，start/stop省略表示全部
  • step 为正数：从左到右，从 start 开始
  • step 为负数：从右到左（反向），start 省略从末尾开始
  • 负数下标：-1 是最后一个，-2 倒数第二个
常考案例
lst = [1,2,3,4,5,6]
lst[::2]   → [1,3,5]   # 下标0,2,4
lst[1:4:2] → [2,4]     # 下标1,3
lst[::-1]  → [6,5,4,3,2,1] # 反转
lst[0:-2]  → [1,2,3,4] # 去掉最后两个
lst[:]     → [1,2,3,4,5,6] # 完整拷贝

# s[1:4:2] 步长2，取下标1,3
s = ["seashell","gold","pink","brown","purple","tomato"]
s[1:4:2] → ['gold','brown']
切片等价关系
  • s[0:-1] ≡ s[:len(s)-1] ≡ s[:-1]
  • s[-1] ≡ s[len(s)-1]
  • lst[::2] → 取下标为0和偶数的元素

元组 tuple

元组特性 & 易错点

元组核心特性
  • 用圆括号 () 表示，元素间用逗号分隔
  • 不可变：创建后不能增删改元素
  • 可以通过下标访问，支持切片
  • 元素可以是不同类型
  • 一个元组可以作为另一个元组的元素（多级索引）
  • 可用 len()、in、+、*

元组中元素不可以是不同类型 ❌ — 元组元素可以是不同类型
元组是可变数据类型 ❌ — 元组不可变
元组一旦创建就不能修改 ✅
Python中元组采用逗号和括号（可选）来表示 ✅

字典 dict

字典操作全览

字典特性
  • 用花括号 {}，键值对 key: value
  • key 必须是不可变类型（整数、字符串、元组均可，列表❌）
  • value 可以是任意类型
  • 无序，不能用下标索引，用 d[key] 访问
  • 可动态增删键值对
  • len(d) 返回键值对数量
字典方法
操作说明
d[key]取值（key不存在则报错）
d.get(key)取值（key不存在返回None）
d[key]=val添加/修改
del d[key]删除键值对
d.clear()清空字典，保留变量
d.keys()所有键
d.values()所有值
key in d判断key是否存在
常考代码
D = {'A':10,'B':20,'C':30,'D':40}
D['D']    → 40   # 用key访问
D[3]      → 报错 # 不能用下标

d = {'abc':123,'def':456,'ghi':789}
len(d) → 3   # 键值对数量

d = {1:'a', 2:'b', 3:'c'}
len(d) → 3

# 月份花卉字典 - .get()用法
MonthandFlower.get(str(n)+"月")
# n是整数1-12，转成"1月"等格式查询
不能创建字典的情况
# 以下可以创建字典：
d = {}              # 空字典 ✅
d = {(1,2):3, ...}  # tuple作key ✅
d = {'张三':1, ...}  # 字符串作key ✅

# 不能创建字典：
d = {[1,2]:3, ...}  # 列表不能作key ❌
d = {1:[2],3:[4]}   # 这个... key是整数 ✅ 可以
d.clear() 清空字典并保留变量；del d 是删除变量本身
集合 set

集合操作

集合特性
  • 无序，不支持索引和切片
  • 元素唯一，自动去重
  • 空集合必须用 set()，不能用 {}
  • set(列表) 可以去重
集合运算
符号方法含义
s1 | s2s1.union(s2)并集
s1 & s2s1.intersection(s2)交集
s1 - s2s1.difference(s2)差集
常考代码
a = set([1,2,3,3,4,4,4])
# a = {1,2,3,4}，自动去重
sum(a) → 10

# 以下语句不能创建集合：
s3 = {1,2,3,4}  # ← 这其实是集合 ✅
# 等等，{1,2,3,4} 是集合没有冒号
# 不能创建集合的是 {1,2,3,4} 写成集合是对的
# 考题：s3={1,2,3,4} 是集合，不是字典

# 不能创建集合：
s = {[1,2], 3}  # 列表不能放入集合 ❌
易错判断
s1 | s2 表示集合并运算 ✅
s1 & s2 与 s1.intersection(s2) 等价 ✅（交集）
集合是有序的 ❌
s=set() 不是空集合 ❌ — set() 就是空集合 ✅

from math import sin, cos

y = int(input())

三个变量公式
from math import sin, cos

a = int(input())
b = int(input())
c = int(input())

# (a+b+c)/(sin(a)√(a²+b²)) + ∛(a³+b³+c³)
n = (a+b+c) / (sin(a)*(a**2+b**2)**(1/2)) \
    + (a**3+b**3+c**3)**(1/3)
print("%.2f" % n)
公式翻译要点
  • √x → x**(1/2) 或 x**0.5
  • x² → x**2
  • ∛x → x**(1/3)
  • 分数 → 分子用括号包裹 (a+b)/(c+d)
  • sin/cos 需要 from math import sin,cos
  • 输出保留2位小数：print("%.2f"%n)
不要在 input() 里加提示文字，否则不得分
另一道公式
from math import sin, cos
x = int(input())
y = int(input())

# (-x+√(x²+4y))/(2x+1) + sin(y) + cos(x)

    + sin(y) + cos(x)
print("%.2f" % n)

改错常见错误类型
  • 缩进错误：if/else/for/while 下的语句必须缩进
  • 比较运算符：赋值用 =，比较用 ==
  • 逻辑运算符：Python 用 and / or，不用 && / ||
  • break 位置：应在成功条件内，不能在外面
  • range 边界：range(3) 是0,1,2；range(1,4)是1,2,3
  • 类型转换：input()返回字符串，用int()转换
  • else:配合for：for...else，else在循环正常结束后执行
